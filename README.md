# Blocto Provider

## `Setup`

```
yarn
# generate local https cert
brew install mkcert
mkcert -install
cd dev-cert
mkcert localhost
```

## `Scripts`

`build`: build dist
`start`: run live-reload dev server on `https://localhost:7777`

## `Develop`

open browser and navigate to `https://localhost:7777/test.html`
